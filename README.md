# Discord_User_Info

Simple tool for checking user information.
You can use User ID or by It's Token (Using by It's token gives more information)

Library used:
- Newtonsoft.JSON

Make sure you change this line
- wc.Headers.Set("Authorization", "Bot bots-token");
to your Bot's Token

To create a Bot
